Указания по работе с тренажерами

Введенные числа проверяются с точностью 10-4
Чтобы перейти к следующему шагу по нажатию кнопки "Дальше!", должны быть правильно заполнены все поля ввода чисел и формул.

Ввод и редактирование формул

Для изменения формул, которые отображаются в виде кнопок щелкните один раз по ним мышью и введите новый текст формулы.

Формулы записываются по стандартным правилам языков программирования.
Возведение в степень - x2 = x^2

Поддерживаемые функции:

sqrt(x) - корень квадратный
abs(x) - модуль |x|
exp(x) - ex
ln(x) - логарифм натуральный
lg(x) - логарифм десятичный
sh(x) - синус гиперболический
ch(x) - косинус гиперболический
sin(x) - синус
cos(x) - косинус
sin2(x) - sin2 x
cos2(x) - cos2 x
tg(x) - тангенс
ctg(x) - котангенс
sec(x) - секанс : 1/cos(x)
cosec(x) - косеканс : 1/sin(x)
arcsin(x) - арксинус
arccos(x) - арккосинус
arctg(x) - арктангенс
arcctg(x) - арккотангенс
