Вказівки по роботі з тренажерами

Введені числа перевіряються з точністю 10-4, якщо немає інших вказівок в умові завдання.
Щоб перейти до наступного кроку після натискання кнопки "Далі!", повинні бути правильно заповнені всі поля введення чисел і формул.

Введення та редагування формул

Для зміни формул, які відображаються у вигляді кнопок натиснiть один раз по ним мишкою та введіть новий текст формули.

Формули записуються за стандартними правилами мов програмування.
Піднесення до степеня - x2 = x^2

Функції, що пiдтримуються:

sqrt(x) - квадратний корінь
abs(x) - модуль |x|
exp(x) - ex
ln(x) - логарифм натуральный
lg(x) - логарифм десятковий
sh(x) - синус гіперболічний
ch(x) - косинус гіперболічний
sin(x) - синус
cos(x) - косинус
sin2(x) - sin2 x
cos2(x) - cos2 x
tg(x) - тангенс
ctg(x) - котангенс
sec(x) - секанс : 1/cos(x)
cosec(x) - косеканс : 1/sin(x)
arcsin(x) - арксинус
arccos(x) - арккосинус
arctg(x) - арктангенс
arcctg(x) - арккотангенс

При використанні тригонометричних функцій аргумент вводити в радіанах
