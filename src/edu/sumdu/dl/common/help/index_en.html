Instructions on using the trainers

Entered numbers are checked with precision 10-4, if there no any other instructions in the problem statement.
To go to the next step by pressing the "Next!" all the entering number fields and formulas must be correctly filled in .

Entering and editing formulas

To change the formulas that appear as buttons, click once with the mouse and enter the new formula text.

Formulas are written on the standard rules of programming.
Involution - x2 = x^2

Supported functions:

sqrt(x) - square root
abs(x) - module |x|
exp(x) - ex
ln(x) - natural logarithm
lg(x) - decimal logarithm
sh(x) - hyperbolic sine
ch(x) - hyperbolic cosine
sin(x) - sine
cos(x) - cosine
sin2(x) - sin2 x
cos2(x) - cos2 x
tg(x) - tangent
ctg(x) - cotangent
sec(x) - secant : 1/cos(x)
cosec(x) - cosecant : 1/sin(x)
arcsin(x) - arcsine
arccos(x) - arccosine
arctg(x) - arctangent
arcctg(x) - arccotangent

On using trigonometric functions argument should be entered in radians
